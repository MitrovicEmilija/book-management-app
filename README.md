# Book Management App

This repository contains a microservices-based system for managing books. The system includes three backend microservices and a frontend web application.

## Services
- **Book Service**: Manages book data like title, author, and genre.
- **User Service**: Handles user management (authentication, profiles).
- **Transaction Service**: Tracks book transactions like purchases or borrowings.

## Frontend
A React-based web application that serves as the user interface to interact with the backend services.

## Getting Started

### Prerequisites
- Docker (for running services in containers)
- Node.js (for frontend development)

### Running the Project
- Set up each microservice in its respective folder.
- For the frontend, run `npm start` to launch the React app.

For detailed instructions on each service, please check the individual service README files.
